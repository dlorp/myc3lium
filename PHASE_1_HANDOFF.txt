CORE FEATURES
=============

State Machine (7 states):
  1. BOOT - Load config, detect first-boot
  2. CONFIG - Validate configuration
  3. SETUP_MODE - WebUI for initial setup
  4. CONNECT - Join myc3-mesh network
  5. POL_DETECT - Antenna polarization sweep
  6. OPERATIONAL - Normal operation
  7. ERROR - Critical failure

Configuration Storage (NVS):
  - UUID generation on first boot
  - Node identity (name, role, variant)
  - Polarization (mode, current, RSSI, recheck interval)
  - Network (mesh SSID, key, router IP)
  - Power (sleep mode, wake interval, TX power)
