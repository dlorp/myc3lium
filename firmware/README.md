# MYC3LIUM Firmware

ESP32 firmware for MYC3LIUM nodes using HYPHA integration.

## Status

🚧 **Coming Soon** - Firmware development will begin after frontend/backend scaffold is complete.

## Planned Stack

- **Platform:** ESP32 (ESP32-S3 or similar)
- **Framework:** Arduino / ESP-IDF
- **Integration:** HYPHA client for WebRTC mesh networking
- **Sensors:** Environmental sensors (temperature, humidity, etc.)
- **Connectivity:** WiFi + mesh networking via HYPHA
