## 2. DEEP WORK: DTC Timeline Visualizer (03:05-03:55)

### Prototype Goal
Demonstrate **r3LAY Phase 3 concept**: Diagnostic event correlation on maintenance timeline

### Problem Statement
Automotive diagnostics often involve:
- Recurring DTCs after "failed" repairs
- Pattern detection across freeze frame data
- Correlation between service history and code recurrence
- No integrated view of timeline → manual pattern detection

### Solution
Build **DTC Timeline Visualizer** — a TUI + CLI tool that:
1. Renders vehicle maintenance history as ASCII timeline
2. Detects recurring DTCs and failed repair patterns
3. Analyzes freeze frame data for driving condition correlation
4. Exports to multiple formats (JSON, Markdown, CSV)

### Sample Data Validation

**Scenario:** 2002 Subaru Outback EJ22 with recurring P0420 (catalyst efficiency)

**Timeline:**
1. Nov 15 - P0420 detected @ 145,203 mi (highway cruise: 2400 RPM, 62 mph, 78% load)
2. Nov 18 - Replaced upstream O2 sensor ($85)
3. Nov 18 - P0420 cleared
4. Dec 05 - P0420 recurred (17 days later) @ 145,890 mi
5. Dec 10 - Replaced downstream O2 sensor ($90)
6. Dec 10 - P0420 cleared
7. Dec 28 - P0420 recurred again (18 days later) @ 146,800 mi
8. Jan 05 - **Smoke test revealed vacuum leak at intake manifold gasket** ($220)
9. Jan 05 - P0420 cleared
10. Mar 15 - Oil change + inspection — **no DTCs, runs clean**

**Correlation Engine Output:**
```
🔴 CRITICAL: P0420 recurred 3 times
   → Multiple recurrences — likely misdiagnosis or deeper issue

⚠️  WARNING: Failed repair (upstream O2 sensor)
   → Code recurred within 17 days — repair addressed symptom, not root cause

⚠️  WARNING: Failed repair (downstream O2 sensor)
   → Code recurred within 18 days — repair addressed symptom, not root cause

ℹ️  PATTERN: P0420 linked to highway cruise conditions
   → ~2500 RPM, ~65 mph, ~78% load
   → Check for vacuum leaks, lean condition at high load
```

**Real-world value:** Timeline revealed pattern that saved owner from unnecessary catalytic converter replacement ($800+). Root cause was $220 intake manifold gasket, which could have been diagnosed earlier with timeline correlation.

### Key Features Implemented

#### 1. Timeline Renderer
- ASCII timeline with month grouping
- Event icons: ⚠️ (DTC detected), 🔧 (service), ✅ (cleared), 🔴 (recurred)
- Freeze frame data display (RPM, speed, load, temp)
- Cost tracking for service events
- Service notes integration

#### 2. Correlation Engine
**Three pattern types:**
- **RecurringCodePattern** - DTC appears multiple times
- **FailedRepairPattern** - Code recurs after service
- **DrivingConditionPattern** - DTCs correlate with specific conditions

**Severity classification:**
- Critical: 3+ recurrences or <14 day recurrence
- Warning: 2 recurrences or 14-30 day recurrence
- Info: Pattern detected but not chronic

**Recommendations:**
- Context-aware suggestions based on freeze frame analysis
- Highway cruise pattern → "Check vacuum leaks, lean condition at high load"
- Low RPM pattern → "Check idle control, PCV valve"
- Quick recurrence → "Repair addressed symptom, not root cause"
