   - DTC reading (Mode 03)
   - Auto-parsing hex responses

2. Live Sensor Dashboard
   - Engine RPM, vehicle speed, coolant temp
   - Throttle position gauge (0-100%)
   - Fuel level gauge (red warning <20%)
   - 500ms refresh rate

3. Terminal UI
   - ratatui layout (Gauge, List, Paragraph widgets)
   - PSX color scheme (cyan/yellow/red/green)
   - Keyboard shortcuts: q (quit), r (refresh DTC)
